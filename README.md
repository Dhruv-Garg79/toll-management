# Toll Management Application

https://drive.google.com/file/d/1ypSOsak0hrHxmbFshMy4oBtsmtHZo6h7/view

### Tech stack used:

- Node.js 18
- bun as a package manager
- mongodb
- fastify server
- zod for runtime validation

### To run the application

```
bun install

in 1st terminal
bun run compile:watch

in 2nd terminal
bun run dev
```
